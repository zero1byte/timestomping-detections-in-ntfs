About Timestomping Detection

Timestomping is a technique used to manipulate file timestamps to evade forensic detection. This tool compares $STANDARD_INFORMATION and $FILE_NAME attributes in NTFS to identify inconsistencies.

$STANDARD_INFORMATION

User-modifiable timestamps visible to applications.

$FILE_NAME

System-managed timestamps stored in MFT records.

Detection

Flags files where these attributes show discrepancies.
